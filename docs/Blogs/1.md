# 计算机英语

*本文整理了那些容易读错的 / 容易理解错的 / 反复出现的 / 区别于日常英语的计算机专业英语单词 / 短语，以及它们的中文译名，用于复习和记忆，让阅读浏览英语资料变得更加容易*

词条格式：

```
WORD|PHRASE : [FEATURE] MEANING [; [FEATURE] MEANING ...] [/ NOTE ...]
```
